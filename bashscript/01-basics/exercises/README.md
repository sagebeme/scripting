# Exercises - Level 1: Basics

## üìù How to Complete Exercises

1. **Read the exercise file**: Each exercise file has comments explaining what to do
2. **Add your commands**: Replace the comments with actual bash commands
3. **Make executable**: Run `chmod +x exercise-name.sh`
4. **Test your solution**: Run `./exercise-name.sh`
5. **Compare with solutions**: Check the `solutions/` directory (after attempting)

## üìã Exercise List

### Exercise 1: Basic Navigation (`exercise-1-navigation.sh`)
**Objective**: Practice navigating the file system
- Navigate directories
- Create directories
- List directory contents
- Use pwd command

**Time**: 10-15 minutes

### Exercise 2: File Operations (`exercise-2-file-operations.sh`)
**Objective**: Learn file creation, copying, moving, and deletion
- Create files
- Add content to files
- Copy files
- Rename files
- Delete files

**Time**: 15-20 minutes

### Exercise 3: Your First Script (`exercise-3-first-script.sh`)
**Objective**: Write a complete working script
- Add shebang
- Add comments
- Use echo command
- Display system information

**Time**: 10-15 minutes

### Exercise 4: Practice with Commands (`exercise-4-commands.sh`)
**Objective**: Combine multiple commands
- Create directories and files
- Copy and move files
- Remove files and directories
- Practice command chaining

**Time**: 20-25 minutes

## üí° Tips

- **Read error messages**: They tell you what went wrong
- **Test incrementally**: Add one command at a time and test
- **Use man pages**: `man command-name` for help
- **Practice daily**: Repetition helps retention
- **Don't peek at solutions**: Try first, then check
