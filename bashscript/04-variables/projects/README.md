## üí° Tips

- **Use ${}**: Prefer ${var} for clarity and safety
- **Default values**: Use parameter expansion for defaults
- **Quote variables**: Always quote when values might contain spaces
- **Validate input**: Check if variables are set before using
- **Test expansions**: Verify parameter expansion results
- **Document**: Explain complex variable usage
